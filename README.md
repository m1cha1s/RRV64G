# RRV64G
A small RV64G complient RISC-V core implemented in no_std Rust
Mayby will be converted to RV64GC and thus may be renamed to RRV64GC for even more compatibility

## Decoding

### rv64i
- [x] R-type
- [x] I-type
- [x] S-type
- [x] B-type
- [x] U-type
- [x] J-type
